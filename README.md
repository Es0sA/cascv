## Deploying to Netlify

1. Create a new Netlify account at netlify.com
2. From your Netlify dashboard, click **"Add new site" → "Deploy manually"**
3. Drag and drop the entire `cas-cv-builder` folder into the upload area
4. Your site will be live at a random Netlify URL (e.g. `random-name.netlify.app`)
5. You can rename the site in: **Site settings → General → Site name**
